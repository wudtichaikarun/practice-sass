# practice-sass

### install project
```
//clone this project
git clone git@github.com:wudtichaikarun/practice-sass.git

// install node_modules
npm install
```

### compile sass to css
```
// compile sass
npm run compile:sass
```

### auto reload page
```
//install live-sever
npm install live-sever -g
// if Error! : maybe you need to use sudo
sudo npm install live-server -g
//use live-sever
line-server
```
